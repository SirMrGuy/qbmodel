Documentation

What each folder and file in the repository does.

File/Folder Description
aliases Files recording team names that differ from the ones in Groger Ranks
docs This website
tournaments Stats from each tournament, copied from the database
.gitignore Standard gitignore file
README.md Standard README file
logit.R R code to perform logistic regression on win/loss data
logit.csv Raw win/loss data converted from stats
margin.R R code to perform linear regression on margin of victory data
margin.csv Raw margin of victory data converted from stats
pacesim.py (ALPHA) Python 2 simulation of PACE Tier I playoffs based on win probabilities
ranks.txt Groger Scores (from Oct. 29) for each team
stats.py Python 2 code to extract games between teams which have a Groger Score from raw stats
tourneys.txt List of tournaments used
